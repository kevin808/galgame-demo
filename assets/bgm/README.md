# 🎵 雪山庄事件 BGM 使用说明

## BGM文件列表

### 1. 主题曲/标题画面 (01_theme.wav)
- **时长**: 120秒
- **风格**: Score (配乐)
- **情绪**: 神秘、惊险/悬疑/紧张
- **使用场景**: 标题画面、主菜单
- **音量建议**: 标准音量

### 2. 调查场景 (02_investigation.wav)
- **时长**: 120秒
- **风格**: Score (配乐)
- **情绪**: 严肃/沉思、惊险/悬疑/紧张
- **使用场景**: 玩家调查现场、寻找线索时
- **音量建议**: -10dB到-15dB (中等音量)

### 3. 对话/问询场景 (03_dialogue.wav)
- **时长**: 120秒
- **风格**: Score (配乐)
- **情绪**: 惊险/悬疑/紧张、平静/放松
- **使用场景**: 角色对话、问询嫌疑人时
- **音量建议**: -15dB到-20dB (较低音量，不抢对话)

### 4. 真相揭露 (04_reveal.wav)
- **时长**: 89秒
- **风格**: Epic (史诗)
- **情绪**: 震撼/壮丽/宏大、严肃/沉思
- **使用场景**: 揭露真相、指认凶手的高潮时刻
- **音量建议**: -5dB到-10dB (较高音量)

### 5. True End结局 (05_true_end.wav)
- **时长**: 120秒
- **风格**: Easy Listening (轻松聆听音乐)
- **情绪**: 温暖/友善、悲伤、感人
- **使用场景**: 成功破案后的真实结局
- **音量建议**: 标准音量

### 6. Bad End结局 (06_bad_end.wav)
- **时长**: 60秒
- **风格**: Score (配乐)
- **情绪**: 暗黑、严肃/沉思
- **使用场景**: 推理失败的结局
- **音量建议**: -10dB到-15dB

## 在游戏中使用

所有BGM文件已添加到 `assets/asset_map.json` 中，可以通过以下方式访问：

```javascript
// 加载BGM
const bgmPath = assetMap.bgm.theme; // 或其他BGM key

// 播放BGM示例
const bgmAudio = new Audio(bgmPath);
bgmAudio.loop = true; // 循环播放
bgmAudio.volume = 0.5; // 音量调整
bgmAudio.play();
```

## 推荐使用时机

1. **游戏启动**: 01_theme.wav
2. **第一章调查**: 02_investigation.wav
3. **第二章问询**: 03_dialogue.wav
4. **第三章推理**: 04_reveal.wav
5. **True End**: 05_true_end.wav
6. **Bad End**: 06_bad_end.wav

## 注意事项

- 所有音频文件为WAV格式，音质最佳但文件较大
- 建议根据需要转换为MP3格式以减小文件大小
- 循环播放的BGM（调查、对话场景）已设计为无缝循环
- 结局BGM建议只播放一次，不循环
